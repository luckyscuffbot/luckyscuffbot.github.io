Lucky Scuff Bot Commands

Raffle Commands

  • !startraffle ![keyword]

    Starts a new raffle using the provided but not required keyword. (Note: if no keyword is provided your user defined keyword setting will be used [This is configurable via a setting])

    Arugments:

    ![keyword] - The raffle entry keyword in order to join the on going raffle.

    Example: !startraffle !test

    — This will start a raffle that people can join by typing !test in chat

  • !endraffle

    Stops the current on going raffle

  • !draw

    Draws a winner from the current on going raffle. (Note: if there are no known entries in the current raffle when a winner is being selected, then the current raffle will end automatically)

  • !redraw

    Draws a new winner from the current on going raffle. This will replace the current winner that was chosen. (Note: if there are no known entries in the current raffle when a new winner is being selected, then the current raffle will end automatically)

  • !confirm

    Confirms the current winner that is chosen by !draw command. This must be done before LSP can be distributed to everyone who joined the raffle.

  • !undo

    Similiar to !redraw, it will revert the current winner, but does not automatically select a new winner. User must initiate another !draw command to select a new possible winner

LSP Commands

  • !lsp

    Command to check the user’s current LSP percentage

  • !addlsp [@username] [lsp amount]

    Adds amount of LSP to the user specified via the command arguments.

    Arugments:

    [@username] - The chat username that the LSP will be given to. ‘@’ is not required, but the command supports using the built in chat autocomplete functionality

    [lsp amount] - The amount of LSP that will be given to the chat user. (Note: LSP amount will always be considered a positive number. You cannot subtract LSP from a chat user please see !setlsp)

    Example: !addlsp @cool_chat_user01 5

    — This will add 5 LSP to cool_chat_user01

  • !addlsp all [lsp amount]

    Adds amount of LSP to all users who participated in the last raffle

    Arugments:

    [lsp amount] - The amount of LSP that will be given to the users. (Note: LSP amount will always be considered a positive number. You cannot subtract LSP from a chat user please see: !setlsp command)

    Example: !addlsp @cool_chat_user01 5

    — This will add 5 LSP to cool_chat_user01

  • !setlsp [@username] [lsp amount]

    Set the amount of LSP for a user

    Arugments:

    [@username] - The chat username that the LSP will be given to. ‘@’ is not required, but the command supports using the built in chat autocomplete functionality

    [lsp amount] - The amount of LSP that will be set for the chat user. (Note: LSP amount will always be considered a positive number. You cannot subtract LSP from a chat user please see !setlsp)

    Example: !setlsp @cool_chat_user01 5

    — This will set cool_chat_user01 LSP to 5
